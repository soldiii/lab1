Правка

Функция "Отменить":

Данная функция предназначена для отмены предыдущей операции пользователя.

Функция "Повторить":

Данная функция предназначена для возврата предыдущей операции пользователя.

Функция "Вырезать":

Данная функция предназначена для удаления выделенного участка текста.

Функция "Копировать":

Данная функция предназначена для копирования в буфер обмена выделенного участка текста.

Функция "Вставить":

Данная функция предназначена для вставки текста из буфера обмена в область редактирования текста.

Функция "Удалить все":

Данная функция предназначена для удаления всех записей в области редактирования текста.

Функция "Выделить все":

Данная функция предназначена для выделения всех записей в области редактирования текста.
