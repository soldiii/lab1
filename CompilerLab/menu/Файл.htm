Файл

Функция "Создать":

Данная функция предназначена для создания файла в новом окне.

Функция "Открыть":

Данная функция предназначена для открытия файла. Необходимо нажать на данный пункт меню и выбрать файл, который необходимо открыть.

Функция "Сохранить":

Данная функция предназначена для сохранения текущей конфигурации файла.

Функция "Сохранить как":

Данная функция предназначена для сохранения файла в требуемом месте вычислительной машины. Необходимо нажать на данный пункт меню, ввести название сохраняемого файла и выбрать папку на вычислительной машине.

Функция "Выход":

Данная функция предназначена для выхода из программы.
